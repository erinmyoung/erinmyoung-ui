# Spinner

Spinner component with different props.

#### Example

<Canvas of={Spinner.Default} layout="padded" />

## Usage

```ts
import {Spinner} from "erinmyoung-ui";

const SpinnerExample = () => {
  return (
      <Spinner
        size={"small"}
      />
  );
};

export default SpinnerExample;
```

#### Arguments

- **size** - A string with one of three possible values: "small," "medium," or "large." It defines the width and height of the spinner.
