# ImageWithBackground

ImageWithBackground component with different props.

#### Example

<Canvas of={ImageWithBackground.Default} layout="padded" />

## Usage

```ts
import {ImageWithBackground} from "erinmyoung-ui";

const ImageWithBackgroundExample = () => {
  return (
      <ImageWithBackground
        src={"/assets/erin.webp"}
        alt={"Erin Young Image"}
        size={"small"}
      />
  );
};

export default ImageWithBackgroundExample;
```

#### Arguments

- **src** - A string representing the source of the foreground image whether it's an external url or relative path in project.
- **alt** - A string that represents the alt text for the foreground image.
- **size** - A string with one of three possible values: "small," "medium," or "large." It defines the width and height of the background image and foreground image.
