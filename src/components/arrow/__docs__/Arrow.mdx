# Arrow

Arrow component with different props.

#### Example

<Canvas of={Arrow.Right} layout="padded" />

## Usage

```ts
import {Arrow} from "erinmyoung-ui";

const ArrowExample = () => {
  return (
      <Arrow
        $orientation={"right"}
        onClick={()=> console.log("Clicked")}
      />
  );
};

export default ArrowExample;
```

#### Arguments

- **$orientation** - A string with one of two possible values: "right" or "left". It defines the horizontal orientation of the arrow.
- **disabled** - A boolean indicating whether the button should be disabled or not. When disabled, the button cannot be clicked or interacted with.
- **onClick** - A function that is called when the button is clicked. It receives a MouseEventHandler for handling the click event on the button element.
